Qubit Circuit Builder

A visual sandbox for quantum circuits

What is this?

Write quantum circuits using Qiskit Python syntax and instantly see them rendered as interactive diagrams. Supports all standard gates — from single-qubit H, X, Z to multi-qubit CX, CCX, SWAP, and parametric gates like RX(θ).

Write or load code

Type Qiskit code directly into the editor, or choose from five built-in examples — Teleportation, Superdense Coding, GHZ State, Phase Estimation, and Parameterized C-Phase. The syntax highlighting helps you catch errors as you type.

Live preview

The circuit diagram updates in real time as you edit. Drag to pan around the circuit and pinch or scroll to zoom in — up to 6x magnification. Mathematical expressions like θ/π render beautifully with LaTeX.
